How itWorksQuick Guide

Welcome to Blood Route Optimizer 🩸

This tool plans the most time-efficient daily blood-delivery routes from a depot to multiple hospitals, splitting the work across several vehicles. Here's everything you need to know.

🧭 Two views: The app has a ⚙ Setup view (add hospitals & choose depot) and a 📊 Results view (routes & maps). Switch with the toggle at the top — or just press the ← / → arrow keys on your keyboard. It jumps to Results automatically when you optimize.
1

Set your ORS API key (recommended)

Open ⚙ Settings and paste a free OpenRouteService key. Without it, the app can only estimate with straight lines and can't look up real hospital locations. Get one free at openrouteservice.org → Sign up → Dashboard → Request a token. Settings is also where you set stop time per hospital, max hospitals per car, departure time (used for arrival estimates), and workload balancing.

2

Pick a starting depot

Type any starting location — the location box has autocomplete too, or leave it blank to default to ศูนย์บริการโลหิตแห่งชาติ สภากาชาดไทย.

3

Add your hospitals

Adding is fully manual — there's no automatic name lookup, so accuracy is entirely in your hands:

  • 📍 Add hospital — type the name, an optional district, and its exact latitude/longitude (paste "lat, lng" copied straight from Google Maps works too).

Each hospital shows a status badge — normally ✓ geocoded once you've entered it. If it lands far from the rest of your group it gets flagged ⚠ far from others as a nudge to double-check the coordinates you typed.

4

Fix any wrong locations 📍

Every hospital has a 📍 Fix button (also clickable via its status badge). It opens a small box where you type the exact latitude/longitude — or paste a "lat, lng" pair copied from Google Maps — then Save. Use this whenever a hospital is flagged or lands in the wrong place — it guarantees accuracy.

5

Save & reuse your lists 💾

Use the 💾 Save button above the hospital list to store the current set under a name (e.g. "Monday route"). Reload it anytime from the dropdown — lists persist across page refreshes and sessions, so you don't have to re-enter hospitals each day.

6

Optimize routes ▶

Press Optimize routes. The app then:

  • Clusters hospitals into groups (one per car) using K-Means, balanced by driving workload so no driver is overloaded.
  • Finds the fastest visiting order for each car — minimizing total driving time, not just distance.
  • Builds a two-phase journey for every car. Hospitals with no valid location are skipped automatically.
7

Understand the two phases

Each car runs a round trip in two phases:

Phase 1 · Collect — Depot → hospitals (pick up empty blood containers) → ศูนย์บริการโลหิตแห่งชาติ สภากาชาดไทย (fill containers with blood).
Phase 2 · Deliver — TRCS → hospitals (deliver the filled blood) → back to depot.

Both phases visit the same hospitals, but each is optimized separately for its own start point, so the order can differ.

8

Read the results

Each car gets a card. Use the Phase 1 / Phase 2 toggle on the card to switch between the collect and deliver legs — the stop list, route flow, and time/distance update accordingly. Each stop shows its estimated 🕐 arrival time, based on the departure time you set in Settings plus driving and stop times. The header shows total stops and estimated full-trip time. Tap Open in Google Maps to navigate the full journey on a phone.

9

Explore the maps

Two side-by-side maps show the Phase 1 and Phase 2 routes following real roads. Use the car filter on each map to focus on a single vehicle, or view all at once. The header also has a dark-mode toggle (🌙) and a language switch (🇬🇧 EN / 🇹🇭 ไทย).

💡 Accuracy tip: Since adding is manual, double-check the coordinates you enter — copy them straight from Google Maps for the best accuracy. Watch for the ⚠ flag and use 📍 Fix to correct anything that looks off.

ยินดีต้อนรับสู่ Blood Route Optimizer 🩸

เครื่องมือนี้ช่วยวางแผนเส้นทางส่งเลือดประจำวันให้มีประสิทธิภาพด้านเวลามากที่สุด จากจุดเริ่มต้นไปยังโรงพยาบาลหลายแห่ง โดยแบ่งงานให้รถหลายคัน ต่อไปนี้คือทุกสิ่งที่คุณควรรู้

🧭 สองมุมมอง: แอปมีมุมมอง ⚙ ตั้งค่าข้อมูล (เพิ่มโรงพยาบาลและเลือกจุดเริ่มต้น) และ 📊 ผลลัพธ์ (เส้นทางและแผนที่) สลับด้วยปุ่มด้านบน หรือกด ปุ่มลูกศร ← / → บนคีย์บอร์ด และจะข้ามไปหน้าผลลัพธ์อัตโนมัติเมื่อจัดเส้นทางเสร็จ
1

ตั้งคีย์ ORS API (แนะนำ)

เปิด ⚙ ตั้งค่า แล้ววางคีย์ OpenRouteService ฟรี หากไม่มีคีย์ แอปจะประมาณได้แค่ระยะเส้นตรงและค้นหาตำแหน่งจริงของโรงพยาบาลไม่ได้ ขอคีย์ฟรีที่ openrouteservice.org → สมัคร → Dashboard → ขอ token ในหน้าตั้งค่ายังกำหนดเวลาต่อโรงพยาบาล จำนวนโรงพยาบาลสูงสุดต่อรถ เวลาออกเดินทาง (ใช้ประมาณเวลาถึง) และการกระจายงานได้ด้วย

2

เลือกจุดเริ่มต้น

พิมพ์จุดเริ่มต้นใดก็ได้ — ช่องนี้มีการเติมข้อความอัตโนมัติด้วย หรือเว้นว่างไว้เพื่อใช้ศูนย์บริการโลหิตแห่งชาติ สภากาชาดไทยเป็นค่าเริ่มต้น

3

เพิ่มโรงพยาบาล

การเพิ่มโรงพยาบาลเป็นแบบกรอกเองทั้งหมด — ไม่มีการค้นหาชื่ออัตโนมัติ ความแม่นยำจึงขึ้นอยู่กับคุณ:

  • 📍 เพิ่มโรงพยาบาล — พิมพ์ชื่อ เขต (ไม่บังคับ) และละติจูด/ลองจิจูดที่แน่นอน (วาง "lat, lng" ที่คัดลอกจาก Google Maps ก็ได้)

โรงพยาบาลแต่ละแห่งจะแสดงป้ายสถานะ — ปกติจะเป็น ✓ geocoded เมื่อกรอกเสร็จ หากอยู่ไกลจากกลุ่มผิดปกติจะถูกทำเครื่องหมาย ⚠ far from others เพื่อเตือนให้ตรวจสอบพิกัดที่กรอกอีกครั้ง

4

แก้ไขตำแหน่งที่ผิด 📍

โรงพยาบาลทุกแห่งมีปุ่ม 📍 แก้ไข (กดที่ป้ายสถานะก็ได้) กดเพื่อเปิดกล่องเล็กๆ ให้พิมพ์ละติจูด/ลองจิจูดที่ถูกต้อง หรือวาง "lat, lng" จาก Google Maps แล้วกดบันทึก ใช้เมื่อโรงพยาบาลถูกทำเครื่องหมายหรืออยู่ผิดที่ — รับประกันความแม่นยำ

5

บันทึกและใช้รายการซ้ำ 💾

ใช้ปุ่ม 💾 บันทึก เหนือรายการโรงพยาบาลเพื่อเก็บชุดปัจจุบันไว้ในชื่อ (เช่น "เส้นทางวันจันทร์") โหลดกลับมาได้ทุกเมื่อจากเมนูแบบเลื่อนลง — รายการจะคงอยู่แม้รีเฟรชหน้าหรือปิดเปิดใหม่ จึงไม่ต้องกรอกโรงพยาบาลใหม่ทุกวัน

6

จัดเส้นทาง ▶

กด จัดเส้นทาง จากนั้นแอปจะ:

  • จัดกลุ่ม โรงพยาบาลเป็นกลุ่ม (หนึ่งกลุ่มต่อรถหนึ่งคัน) ด้วย K-Means โดยสมดุลตามภาระการขับ ไม่ให้คนขับคนใดงานหนักเกินไป
  • หาลำดับการเยี่ยมที่เร็วที่สุด ของแต่ละคัน — ลดเวลาขับรวม ไม่ใช่แค่ระยะทาง
  • สร้างการเดินทางสองเฟสสำหรับทุกคัน โรงพยาบาลที่ไม่มีตำแหน่งจะถูกข้ามอัตโนมัติ
7

เข้าใจสองเฟส

รถแต่ละคันเดินทางไปกลับเป็นสองเฟส:

เฟส 1 · เก็บภาชนะ — จุดเริ่มต้น → โรงพยาบาล (รับภาชนะเลือดเปล่า) → ศูนย์บริการโลหิตแห่งชาติ สภากาชาดไทย (เติมเลือดลงภาชนะ)
เฟส 2 · ส่งเลือด — ศูนย์บริการโลหิตแห่งชาติ สภากาชาดไทย → โรงพยาบาล (ส่งเลือดที่เติมแล้ว) → กลับจุดเริ่มต้น

ทั้งสองเฟสเยี่ยมโรงพยาบาลชุดเดียวกัน แต่แต่ละเฟสถูกจัดให้เหมาะกับจุดเริ่มต้นของตัวเอง ลำดับจึงต่างกันได้

8

อ่านผลลัพธ์

รถแต่ละคันจะมีการ์ด ใช้ปุ่มสลับ เฟส 1 / เฟส 2 บนการ์ดเพื่อสลับระหว่างช่วงเก็บและช่วงส่ง — รายการจุด เส้นทาง และเวลา/ระยะทางจะเปลี่ยนตาม แต่ละจุดแสดง 🕐 เวลาถึงโดยประมาณ คำนวณจากเวลาออกเดินทางที่ตั้งไว้บวกเวลาขับและเวลาแวะ ส่วนหัวการ์ดแสดงจำนวนจุดและเวลารวมทั้งทริป กด เปิดใน Google Maps เพื่อนำทางทั้งเส้นทางบนมือถือ

9

สำรวจแผนที่

แผนที่สองฝั่งแสดงเส้นทางเฟส 1 และเฟส 2 ตามถนนจริง ใช้ตัวกรองรถบนแต่ละแผนที่เพื่อดูเฉพาะคันเดียว หรือดูทั้งหมดพร้อมกัน ส่วนหัวยังมีปุ่มสลับโหมดมืด (🌙) และสลับภาษา (🇬🇧 EN / 🇹🇭 ไทย)

💡 เคล็ดลับความแม่นยำ: เนื่องจากการเพิ่มโรงพยาบาลเป็นแบบกรอกเองทั้งหมด ควรตรวจสอบพิกัดที่กรอกให้ถูกต้อง — คัดลอกจาก Google Maps โดยตรงจะแม่นยำที่สุด สังเกตป้าย ⚠ และใช้ 📍 แก้ไข เพื่อแก้สิ่งที่ดูผิดปกติ
